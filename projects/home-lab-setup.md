# Home Lab Setup

## Objective
To create a basic home lab environment for learning cybersecurity fundamentals and practicing system administration in a safe and controlled setup.

## Lab Environment
- Host Machine: Personal computer
- Virtualization Tool: VirtualBox
- Guest Operating System: Ubuntu Linux

## Setup Steps
1. Installed VirtualBox on host machine
2. Created a Linux virtual machine
3. Configured basic system settings
4. Updated the system and installed essential tools

## Learning Outcome
This lab helped me understand how virtual machines work and provided a safe environment to practice basic Linux commands and security concepts.

## Skills Demonstrated
- Virtualization basics
- Linux fundamentals
- System setup and configuration
