# Phishing Awareness & Analysis

## Objective
To understand how phishing attacks work and how to identify common warning signs in suspicious emails.

## Learning Source
- Introductory cybersecurity training materials
- Public phishing examples used for educational purposes

## Key Red Flags Identified
- Urgent or threatening language
- Requests for sensitive or personal information
- Suspicious sender addresses or domains
- Unexpected attachments or links

## Outcome
This project improved my ability to recognize phishing attempts and understand basic attacker techniques used to target users.

## Skills Demonstrated
- Cybersecurity awareness
- Risk identification
- Documentation and reporting
